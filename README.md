## Dependencias

- python 3
- Django
- sqlite
- playwright
- ruff

## Instalar dependencias

```
pip install -r requirements.txt
```

## Iniciar la Base de Datos

```
./start_db.sh
```

### Llenar la base de datos

```
./load_data.sh
```

## Iniciar app

```
python manage.py runserver
```
